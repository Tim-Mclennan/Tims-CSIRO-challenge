CSIRO Web Solutions Junior Web Developer Recruitment Task

This task is provided to you to complete in your own time before your interview. The task is a basic javascript counter, with the following functionality:

  • The box labelled "increment" is for the user to input a number which will be the increment/decrement the counter uses
  • Pressing the button "Add count" will add the increment number to the count
  • Pressing the button "Decrease count" will subtract the increment number from the count

You have been provided with the files required to complete the task, it should take 30 minutes to 1 hour to complete. We expect a fully functional counter, not a prototype, so be mindful of what you need to do to achieve this.
